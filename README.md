# ⚙️ Suby - Food Delivery App Backend

Welcome to the backend repository of **Suby**, a full-stack food delivery platform inspired by Swiggy! 🚀

This backend handles authentication, restaurant management, menu operations, order processing, and secure API communication.

## 📚 Project Overview

Built with **Node.js, Express.js, and MongoDB**, the backend follows a scalable architecture to support users, vendors, and food ordering workflows.
